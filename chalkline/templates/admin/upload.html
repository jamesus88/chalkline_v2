Step 1: Important Notes.

  • Download the template above and edit with your league's schedule.
  • Uploaded schedules must follow the template .csv format.
  • Failure to match format may result in inaccurate uploads and more cleanup.
  • Do not edit the header (first) row. Do not add additional columns.
    • Column Guide:

      The below columns cannot be left empty!

    1. type: "Game" or "Practice". In almost all cases, use "Game"
    2. season: Choose from "Fall", "Spring", or "Summer" followed by a space and the year. e.g. "Fall 2025"
    3. date: yyyy-mm-dd format. e.g. "2025-01-20"
    4. time: Use Military/24-hr time. e.g. "19:30"
    5. length(hrs): duration of the game in hours.
    6. venueId: Unique id of venue game is to be played at. For full list of venues, see Admin - Manage League - Venues
    7. field: field number. e.g. "1"
    8. age: full age group name. e.g. "Majors". Must be exactly one of the age groups listed under Admin - Manage League - Age Groups

      9. The below columns may be left empty!

    9. away: away team code, if any. e.g. "SLL-MAJ1". Must be exactly a team's id from Admin - Teams
    10. home: home team code, if any. e.g. "SLL-MAJ2". Must be exactly a team's id from Admin - Teams
    11. status: status of game, if any. If left blank, default will be "On Time"
    12. locked: mark with "1" to lock game (prevent signups), otherwise leave blank. In almost all cases, leave blank.

      13. Umpire columns:

    13. Position: mark with "1" to create position opening or "0"/"None"/leave empty otherwise
    14. Position_duty: team, if any, responsible for filling such umpire position. e.g. "SLL_MAJ3". Must be exactly a team's id from Admin - Teams
